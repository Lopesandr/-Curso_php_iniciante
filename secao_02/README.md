- Float:  Números com casas decimais, como 1.2, 1.3...


### Como verificar se o tipo de dado e tal...

- Int - `is_int(1)` : Se  retornar True, é inteiro. 
- Float - `is_float(1)` : Se  retornar True, é um float.
